# DevLearn

## Instalation


1. Create Database DevLearn by running the script
2. Setup RabbitMQ locally, run service on default port.
3. Build solution
4. Run Admin site, add some events, and scores.
5. Run State Server app

Start services: 

```
dotnet run
```
